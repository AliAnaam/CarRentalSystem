# CarRentalSystem

This project is a desktop-based application designed to streamline the operations of a car rental agency. It manages vehicle inventory, tracks customer information, and processes rental transactions using a Graphical User Interface (GUI).

Built with Java Swing and NetBeans GUI Builder

---

## Features
- Manage cars: add, update, and delete car records  
- Manage customers: add, edit, and delete customer information  
- Handle rental bookings and track car availability  
- Calculate rental costs based on the number of rental days  
- User-friendly interface using Java Swing with interactive forms and tables

---

## Technologies Used
- Java (Object-Oriented Programming)  
- Java Swing (JFrame, JPanel, JButton, JTable)  
- NetBeans IDE (Drag & Drop GUI Builder)

---

## How to Run
1. Open the project in **NetBeans**  
2. Build the project  
3. Run the `Main` class to launch the application
